# 4. 实验设计方案

## 4.1 研究问题与假设

### 4.1.1 研究问题

基于行为运营管理（Behavioral Operations Management）理论与人机协同决策范式，本研究在四级供应链仿真环境下，围绕情绪扰动、激励机制与协同机制三条主线，提出以下三个递进式研究问题：

- **RQ1**（情绪放大效应）：在动态供需环境下，决策者的损失厌恶倾向如何通过情绪演化机制被放大，进而加剧牛鞭效应？
- **RQ2**（激励阻断机制）：库存精准匹配正向激励函数能否改变深度强化学习智能体的优化目标，从而在系统层面阻断恐慌情绪的逐级蔓延？
- **RQ3**（协同鲁棒性）：多智能体信息共享与协同决策机制在应对突发供应链中断时，相较于单智能体方案是否具有显著的鲁棒性优势？

上述三个问题构成了"**情绪扰动→激励机制→协同鲁棒**"的递进因果链，对应论文的三个核心论点。

### 4.1.2 研究假设

基于上述研究问题，提出以下三个可证伪假设：

- **H1（损失厌恶放大假设）**：动态环境中的缺货事件通过情绪演化方程将决策者情绪状态推向恐慌饱和（$E_t \to -1$），使有效缺货惩罚权重放大约70%，导致订货决策偏离理性最优，加剧牛鞭效应。
- **H2（正向激励阻断假设）**：库存精准匹配正向激励函数通过将深度Q网络（DQN）[6]的优化目标从"最小化缺货"转变为"最大化精准匹配"，引导订货量趋向需求均值，在系统层面阻断"缺货→恐慌→过度订货→上游缺货→更大恐慌"的恶性循环，使分销商BWE降低80%以上。
- **H3（协同鲁棒性假设）**：多智能体协同信息共享机制使上游节点能够提前感知下游需求突变，相较于单智能体方案，制造商BWE进一步降低45%以上，系统平均服务水平提升至99%以上，且在情绪感知噪声干扰下通过弹性权重巩固（EWC）[7]与优先经验回放（PER）[8]机制仍保持稳定。

## 4.2 实验框架设计

### 4.2.1 供应链结构

本研究采用Sterman[1]设计的经典啤酒游戏（Beer Distribution Game）模式，构建四级串联供应链：

```
顾客需求 → 零售商(k=1) → 批发商(k=2) → 分销商(k=3) → 制造商(k=4)
```

每个周期$t$各节点依次经历三个事件：（1）收货——接收上游节点在$L$个运输周期前发出的货物；（2）决策——基于本期需求与库存状态向上游补货；（3）发货——满足下游订单并更新库存与缺货累积。该结构既保留了经典牛鞭效应研究的可比性，又为引入智能体决策与情绪机制提供了可扩展的节点接口。

### 4.2.2 需求生成机制

顾客需求遵循一阶自回归过程AR(1)：

$$D_t = d + \rho \cdot D_{t-1} + \varepsilon_t, \quad \varepsilon_t \sim N(0, \sigma_\varepsilon^2)$$

其中参数取值为：$d = 10$，$\rho = 0.5$（$|\rho| < 1$保证平稳性），$\sigma_\varepsilon = 5$，初始需求$D_0 = d/(1-\rho) = 20$。该参数配置与Chen等[2]的经典研究及李勇等[5]的智慧决策机器人研究保持一致，便于跨研究对比。AR(1)过程既反映了真实需求的自相关性，又为动态突发事件提供了可控的扰动基底。

### 4.2.3 四组对比实验配置

为系统验证三个假设并精确分离各干预要素的独立效应，本研究采用控制变量法设计四组对比实验（含一组剥离实验Exp_1b），各组仅在目标干预要素上存在差异，其余环境参数严格一致：

**表4-1. 四组对比实验配置**

| 实验组 | 设计要素 | 智能体部署 | 情绪模块 | 协同机制 | 动态事件 | 仿真周期 | 验证假设 |
|:------:|:--------:|:----------:|:--------:|:--------:|:--------:|:--------:|:--------:|
| Baseline | 纯理性决策基线 | 无 | 禁用 | 无 | 无 | 20000 | 对照基准 |
| Exp_1 | 单智能体IDMR | 仅分销商(k=3) | 禁用 | 无 | 无 | 20000 | H2 |
| Exp_1b | 单智能体IDMR+情绪 | 仅分销商(k=3) | 启用 | 无 | 无 | 20000 | H1 |
| Exp_2 | 多智能体+情绪+协同 | 全节点 | 启用 | 启用 | 76次 | 20000 | H1, H3 |

四组实验均在相同的AR(1)需求序列（seed=42）、相同的运输延迟（$L=2$）与相同的初始库存（$NS_0 = 10.0$）下运行，确保组间差异仅来源于干预要素本身。全部实验经过20000个周期的连续动态订货与发货，与李勇等[5]的仿真周期数完全对标。其中：

- **Baseline**：四个节点均采用理性决策（SMA移动平均预测+OUT订至点策略），作为牛鞭效应的基准参照；
- **Exp_1**：仅分销商（k=3）部署IDMR智慧决策机器人[5]（DQN[6]+人机协同），其余节点保持理性决策，用于隔离正向激励函数的独立效应。Exp_1关闭情绪模块与正向激励机制，采用纯满足率奖励函数，惩罚阈值设为经典供应链分销商实际平均库存（经2000步预热仿真测得为352.51）；
- **Exp_1b**（剥离实验）：在Exp_1基础上仅启用情绪演化模块与库存精准匹配正向激励机制，不引入多智能体协同通信与动态突发事件，用于隔离情绪机制的独立效应。Exp_1b与Exp_1的唯一差异为情绪模块开关（`enable_emotion=True`且`enable_inventory_bonus=True`），与Exp_2的差异为协同机制与动态事件开关，构成"Exp_1→Exp_1b→Exp_2"的递进式剥离对比链；
- **Exp_2**：全节点部署智能体，启用情绪演化方程与多智能体协同通信通道，并注入76次动态突发事件，用于验证情绪放大、激励阻断与协同鲁棒性的综合效应。

**剥离实验对比逻辑**。通过四组实验的成对比较，可精确分解各干预要素的独立效应与交互效应：

- **情绪机制独立效应**（Exp_1 vs Exp_1b）：两组均为单智能体、无协同、无动态事件，唯一差异为情绪模块开关，可隔离情绪演化对订货决策的独立影响；
- **协同通信独立效应**（Exp_1b vs Exp_2）：两组均启用情绪模块，差异在于协同机制与动态事件，可隔离协同通信对上游BWE与服务水平的独立贡献；
- **情绪+协同联合效应**（Exp_1 vs Exp_2）：综合对比可量化两种机制叠加的系统级效应。

### 4.2.4 IDMR智慧决策机器人设计

IDMR（Intelligent Decision-Making Robot）是基于DQN[6]的人机协同智慧决策机器人，其核心在于将人类理性决策的"最优经验"融入强化学习框架，通过"传授经验—限制决策—惩罚机制"三位一体的人机协同机制，引导智能体在供应链动态环境中学到最优订货策略。

**（1）状态空间**。IDMR在每个周期$t$观察分销商节点的5维状态向量：

$$s_t = \left[ S_{t-1}^3, \; WIP_{t-1}^3, \; q_{t-1}^2, \; Trans_{t-2}^3, \; q_{t-1}^3 \right]$$

其中$S_{t-1}^3$为分销商上期净库存，$WIP_{t-1}^3$为在途库存，$q_{t-1}^2$为批发商上期订单（下游需求信号），$Trans_{t-2}^3$为分销商前期到货量，$q_{t-1}^3$为分销商上期订货量。各维度分别除以缩放因子（库存类÷100，订单类÷50）进行归一化，使值域落入$[0, 1]$区间。

**（2）动作空间**。IDMR的动作为离散订货量$a_t \in [11, 40]$，共30个离散动作，覆盖了AR(1)需求均值（$D_0 = 20$）附近的合理订货范围。

**（3）DQN网络结构**。采用两层全连接网络（$5 \to 64 \to 64 \to 30$），ReLU激活，He初始化，Adam优化器。维护值网络与目标网络，目标网络每10步同步一次。DQN超参数如表4-2所示。

**表4-2. DQN超参数配置**

| 参数 | 符号 | 取值 | 说明 |
|:----:|:----:|:----:|:------|
| 学习率 | $\eta$ | $10^{-4}$ | Adam优化器 |
| 折扣因子 | $\gamma$ | 0.9 | 未来奖励折扣 |
| 批量大小 | $B$ | 32 | 经验回放采样 |
| 回放池容量 | $N$ | 20000 | 经验存储上限 |
| 探索率初始 | $\varepsilon_0$ | 1.0 | ε-greedy初始 |
| 探索率终止 | $\varepsilon_{\min}$ | 0.01 | ε-greedy终止 |
| 目标网络更新频率 | — | 10步 | 软更新间隔 |
| 隐藏层维度 | — | 64-64 | 两层全连接 |

### 4.2.5 人机协同机制

IDMR的人机协同机制包含三个层次，模拟"人类导师—机器学徒"的知识传递与行为约束关系：

**（1）传授经验**。理性决策（SMA+OUT）作为"老师"的决策经验，通过奖励函数中的满足率激励间接传授给IDMR。IDMR的优化目标是最大化服务水平，与理性决策"最大化SL"的目标一致，但IDMR通过DQN的试错学习，能够发现比固定OUT策略更优的动态订货策略。

**（2）限制决策**。采用ε-greedy探索策略，在训练初期（$\varepsilon=1.0$）充分探索动作空间$[11, 40]$，随着训练推进线性衰减至$\varepsilon_{\min}=0.01$，逐渐转向利用学到的最优策略。动作范围$[11, 40]$的设定避免了极端订货量，确保智能体在合理范围内决策。

**（3）惩罚机制**。当IDMR的积压库存达到相同需求下经典多级供应链对应节点（分销商）的平均库存时，决策"老师"禁止IDMR向上游订货。惩罚阈值定义为：

$$\text{Threshold} = \bar{NS}_{\text{classical}}^{(3)} \times \delta$$

其中$\bar{NS}_{\text{classical}}^{(3)}$为经典供应链分销商的实际平均库存，经2000步预热仿真测得为352.51；$\delta$为惩罚倍数，Exp_1中设为1.0（即达到经典平均库存时禁止订货），与李勇等[5]的惩罚机制设计一致。当$NS_t^{(3)} > \text{Threshold}$时，强制$a_t = 0$（禁止订货），防止库存无界囤积。

### 4.2.6 奖励函数设计

IDMR的奖励函数综合了服务水平收益、库存精准匹配正向激励、缺货惩罚与库存积压软惩罚四个分量：

$$r_t = \underbrace{r_{\text{fill}}}_{\text{服务水平}} + \underbrace{r_{\text{bonus}}}_{\text{正向激励}} - \underbrace{r_{\text{stockout}}}_{\text{缺货惩罚}} - \underbrace{r_{\text{holding}}}_{\text{积压惩罚}}$$

各分量定义如下：

**（1）服务水平收益**：

$$r_{\text{fill}} = \frac{\text{fulfilled}_t}{D_t}$$

即当期满足率，直接激励IDMR最大化服务水平，与李勇等[5]公式11的设计思想一致。

**（2）库存精准匹配正向激励**（Exp_2启用，Exp_1禁用）：

$$r_{\text{bonus}} = w_b \cdot \max\left(0, \; 1 - \frac{|NS_t / \hat{D}_{t+1} - 1|}{\Delta_{\max}}\right), \quad \text{if } \theta_L \leq \frac{NS_t}{\hat{D}_{t+1}} \leq \theta_U$$

其中$\hat{D}_{t+1}$为基于SMA（窗口$p=5$）的下期需求预测，$\theta_L = 0.8$与$\theta_U = 1.5$为库存覆盖比例的上下界，$w_b = 0.3$为正向激励权重，$\Delta_{\max}$为最大偏离度。该机制对冲了"损失厌恶"导致的过度订货倾向：当库存精准覆盖预测需求时给予钟形奖励，引导DQN从"最小化缺货"转向"最大化精准匹配"。

**（3）缺货刚性惩罚**：

$$r_{\text{stockout}} = w_s \cdot \frac{\text{stockout}_t}{D_t}$$

其中$w_s = 1.0$为缺货惩罚权重。在情绪模块启用时（Exp_2），该权重受情绪状态调节（见4.2.7节）。

**（4）库存积压软惩罚**：

$$r_{\text{holding}} = w_h \cdot \max(0, NS_t)$$

其中$w_h = 10^{-4}$为极小的积压惩罚权重，仅防止库存无界增长，不对正常库存水平产生显著干扰。

**Exp_1配置**：关闭正向激励（$r_{\text{bonus}} = 0$）和情绪模块，采用纯满足率奖励函数$r_t = r_{\text{fill}} - r_{\text{stockout}} - r_{\text{holding}}$，与李勇等[5]的奖励设计完全一致，确保Baseline对标的有效性。

### 4.2.7 情绪演化方程设计

为模拟人类决策者的心理状态对订货决策的影响，本研究设计了基于tanh饱和动力学的情绪演化模块。

**（1）情绪状态定义**。情绪状态$E_t \in [-1, 1]$，其中$E_t < 0$表示恐慌/焦虑（损失厌恶，倾向过度订货），$E_t > 0$表示自信/乐观（风险中性，倾向精准订货），$E_0 = 0$为初始中性状态。

**（2）情绪演化方程**：

$$E_t = \tanh\left(\alpha \cdot E_{t-1} + \gamma \cdot \Phi_t\right)$$

其中$\alpha = 0.7$为情绪惯性系数（$\alpha$越大情绪越持久），$\gamma = 2.0$为情绪敏感度（对反馈的响应强度），$\Phi_t$为反馈信号。tanh函数将情绪自然限制在$[-1, 1]$区间，连续缺货时$E_t$加速趋向$-1$（恐慌饱和），连续精准匹配时趋向$+1$（过度自信）。

**（3）反馈信号**：

$$\Phi_t = -w_s^{\Phi} \cdot \text{stockout\_rate}_t + w_m^{\Phi} \cdot \text{match\_factor}_t - w_e^{\Phi} \cdot \text{excess\_rate}_t$$

其中$w_s^{\Phi} = 1.0$（缺货→恐慌，负向）、$w_m^{\Phi} = 0.5$（精准匹配→乐观，正向）、$w_e^{\Phi} = 0.3$（过度积压→焦虑，负向）。缺货率$\text{stockout\_rate} = \text{stockout}_t / D_t$，精准匹配度$\text{match\_factor} \in [0, 1]$（1=完美匹配），过度积压率$\text{excess\_rate} = \max(0, (NS_t - \theta_U \hat{D}_{t+1}) / \hat{D}_{t+1})$。

**（4）决策映射（情绪调节奖励权重）**。情绪状态通过调节奖励函数权重影响DQN的优化方向：

$$w_s^{\text{eff}} = w_s \cdot (1 + |E_t|), \quad E_t < 0 \quad \text{(恐慌放大缺货惩罚)}$$
$$w_b^{\text{eff}} = w_b \cdot (1 + E_t), \quad E_t > 0 \quad \text{(乐观放大正向激励)}$$

当恐慌状态（$E_t < 0$）时，有效缺货惩罚权重放大约70%（$E_t = -0.7$时$w_s^{\text{eff}} = 1.7 w_s$），模拟损失厌恶导致的过度订货倾向；当乐观状态（$E_t > 0$）时，有效正向激励权重放大，引导智能体更积极地追求精准匹配。

## 4.3 变量定义与测量

### 4.3.1 自变量

本研究的自变量为实验组的干预要素，包括三个二分变量：

1. **决策方式**（Decision Mode）：理性决策（SMA+OUT）vs 智慧决策（IDMR-DQN）
2. **情绪模块**（Emotion Module）：禁用 vs 启用
3. **协同机制**（Coordination Mechanism）：单智能体 vs 多智能体协同

### 4.3.2 因变量

因变量包括主指标与辅助指标两个层次：

**主指标**（用于假设检验）：

- **牛鞭效应BWE**：$BWE_k = var(q_k)/var(D)$，衡量节点$k$的订单方差相对顾客需求方差的放大倍数
- **服务水平SL**：$SL_k = \frac{1}{T}\sum_{t=1}^{T}\min(1, fulfilled_{k,t}/D_{k,t})$，衡量节点$k$的需求满足率
- **平均成本**：$Cost_k = \frac{1}{T}\sum_{t=1}^{T}\left[h \cdot \max(0, NS_{k,t}) + b \cdot \max(0, D_{k,t} - fulfilled_{k,t})\right]$，其中$h=1.0$为单位库存成本，$b=2.0$为单位缺货成本

**辅助指标**（用于机制分析）：

- **情绪波动指数**：$\sigma_E = std(E_t)$，衡量决策者情绪状态的波动程度
- **过度订货比例**：订货量超过需求均值的周期占比
- **恢复时间**：动态事件后系统恢复至稳态所需的周期数

## 4.4 动态事件设计

为验证H3（协同鲁棒性假设），在Exp_2中注入两类动态突发事件，共计76次：

### 4.4.1 需求突变事件

需求突变模拟终端市场的非预期波动，共53次，分为两种强度：

- **正向需求冲击**（2×）：顾客需求在某周期临时放大至基准的2倍
- **负向需求冲击**（0.5×）：顾客需求在某周期临时缩减至基准的0.5倍

需求突变直接冲击零售商节点，并通过订单逐级向上游传递，用于检验多智能体协同信息共享能否使上游节点提前调整产能。

### 4.4.2 供应中断事件

供应中断模拟上游产能的非预期停摆，共23次，持续3~5个周期：

- 中断期间，被中断节点的发货能力降为零，订单累积至恢复后处理
- 中断位置随机分布于分销商与制造商节点

供应中断直接冲击上游节点，用于检验多智能体框架在恢复过程中维持业务连续性的能力。

### 4.4.3 情绪传染机制

在Exp_2中，情绪状态通过供应链网络以30%的概率向上游传染：

$$E_{k,t}^{\text{upstream}} \leftarrow E_{k,t}^{\text{upstream}} + \eta \cdot (E_{k-1,t} - E_{k,t}^{\text{upstream}}), \quad \eta \sim Bernoulli(0.3)$$

其中$E_{k-1,t}$为下游节点$k-1$在周期$t$的情绪状态。该机制使得零售商的恐慌情绪能够逐级蔓延至批发商、分销商与制造商，为验证正向激励函数的阻断效应提供了可控的扰动通道。

### 4.4.4 持续学习任务切换设计

为验证H3中EWC[7]与PER[8]机制对灾难性遗忘的抑制效果，设计任务切换场景：

- **Task1**（平稳需求）：$\rho=0.5$的AR(1)需求，运行10000周期
- **Task2**（反向剧烈波动）：$\rho=-0.5$的AR(1)需求，$\sigma_\varepsilon=10$，运行10000周期

通过对比有/无EWC、有/无情绪感知噪声（$\sigma_{\text{noise}}=0.1$）三组条件下服务水平、奖励与BWE遗忘率的变化，验证持续学习机制的鲁棒性。情绪感知噪声模型为$E_{\text{perceived}} = \text{clip}(E_{\text{true}} + \mathcal{N}(0, \sigma_{\text{noise}}), -1, 1)$。

## 4.5 数据采集与分析方法

### 4.5.1 数据采集

仿真过程逐周期记录以下三类数据，20000周期×4节点共计约32000条节点级记录：

1. **节点级数据**：各节点的订货量$q_{k,t}$、接收需求$D_{k,t}$、净库存$NS_{k,t}$、在途库存$WIP_{k,t}$、情绪状态$E_{k,t}$、服务水平和成本
2. **系统级数据**：各节点BWE、系统平均SL、系统总成本、情绪波动指数$\sigma_E$
3. **事件级数据**：每次动态事件的类型、强度、持续时间、恢复周期数

### 4.5.2 分析方法

本研究采用以下四种分析方法：

**（1）组间对比与剥离分析**。通过四组实验在主指标（BWE、SL、成本）上的成对差异，量化各干预要素的独立效应与交互效应。具体而言：Exp_1 vs Baseline隔离IDMR智慧决策的效应，Exp_1 vs Exp_1b隔离情绪机制的独立效应，Exp_1b vs Exp_2隔离协同通信与动态事件的独立效应，Exp_1 vs Exp_2量化情绪与协同的联合效应。该剥离实验设计（Ablation Study）通过严格的控制变量法，确保每组比较中仅有一个干预要素发生变化，从而精确归因各机制对牛鞭效应缓解的贡献。

**（2）归因分析**。基于逐周期记录，按情绪状态分组统计订货决策偏差，揭示损失厌恶放大效应的微观机制。例如，对比极端恐慌状态（$E < -0.5$）与中性状态（$-0.1 \leq E \leq 0.1$）下的平均订货量与订货方差，量化情绪对决策的扰动幅度。

**（3）恢复时间分析**。针对76次动态事件，记录系统从扰动恢复至稳态所需的周期数，对比单智能体与多智能体框架在业务连续性上的差异。

**（4）持续学习鲁棒性分析**。基于弹性权重巩固（EWC）[7]与优先经验回放（PER）[8]机制，设计任务切换场景（Task1平稳需求→Task2反向剧烈波动），通过对比有/无EWC、有/无情绪感知噪声三组条件下服务水平、奖励与BWE遗忘率的变化，验证持续学习机制对灾难性遗忘的抑制效果及情绪感知误差下的鲁棒性。

### 4.8.6 假设验证结论

**H1（损失厌恶放大假设）—— 部分验证**。剥离实验Exp_1b证实情绪演化方程确实将决策者情绪推向恐慌饱和（分销商$E=-0.778$，95.9%恐慌占比），且情绪-订货量Pearson相关系数达极显著水平（$p<10^{-60}$）。Exp_2情绪波动指数呈下游放大梯度（$\sigma_E$: 0.59→0.22），零售商SL下降16.16pp验证了情绪放大效应在动态扰动下的显著表现。然而，Exp_1b中情绪机制对BWE的独立效应极小（分销商-1.3%），说明DQN的学习能力可在平稳环境下抵消情绪扰动，情绪放大效应需在动态突发事件触发下才显著显现。

**H2（正向激励阻断假设）—— 验证成立**。分销商BWE从Baseline的67.33降至Exp_2的9.81（-85.43%，超过80%阈值），恐慌时过度订货比例在下游三节点均低于中性状态，证明正向激励成功阻断了恶性循环。剥离实验进一步揭示，Exp_1b（启用情绪+正向激励）分销商BWE为10.51，与Exp_1（10.65）接近，说明正向激励机制在无动态事件时已有效引导DQN趋向精准匹配。

**H3（协同鲁棒性假设）—— 部分验证**。剥离实验精确归因显示，协同通信机制贡献了制造商BWE降幅的绝大部分（Exp_1b→Exp_2: -55.8%，超过45%阈值），而情绪机制独立贡献仅为+0.4%。EWC在无噪声下有效保护旧任务知识。但系统平均SL为94.73%，低于99%阈值，主要受零售商SL（82.76%）拖累——该下降由Exp_2引入的76次动态突发事件与情绪放大效应叠加导致，而非协同机制本身缺陷。
